**日期**: 2026-05-10

## 1. 项目概述

一个手机端优先的趣味娱乐网站，以海绵宝宝比奇堡为主题，包含4个互动小工具。
目标：发给女朋友消遣娱乐，沙雕搞怪中带甜蜜。

## 4. 四个工具规格

### 4.1 🧽 海绵宝宝的夸夸气泡
- **角色**: 海绵宝宝
- **玩法**: 点击"吹泡泡"按钮，气泡从菠萝屋飘出，显示随机夸夸/土味情话
- **视觉**: 气泡形状卡片从下往上弹出，伴随小气泡粒子
- **数据**: 内置30+条夸夸语录数组，随机不重复（连续3次不重复）
- **按钮文案**: "🫧 再吹一个泡泡"
- **示例文案**: "你比蟹老板还有钱！因为你拥有我的心 ❤️"

### 4.2 🐚 神奇海螺的预言
- **角色**: 神奇海螺（Magic Conch Shell）
- **玩法**: 点击海螺/拉绳子，海螺给出今日运势
- **输出**: 运势等级（⭐⭐⭐~⭐⭐⭐⭐⭐）、幸运颜色、今日建议
- **视觉**: 海螺图标，拉绳动画，结果从海螺中"浮现"
- **逻辑**: 基于当天日期 seed 生成确定性的伪随机运势（同一天运势相同）
- **文案风格**: 半真半假，搞笑为主。例："今日不宜打工，宜收红包"

### 4.3 🪼 水母传信（悄悄话密语）
- **角色**: 水母
- **玩法**: 输入文字 → 点击加密 → 生成加密密文 → 复制发给对方 → 对方在此解密
- **加密方式**: Base64 + 简单混淆 + emoji 包装（非真安全，纯趣味）
- **视觉**: 水母图标，电波闪烁动画，加密过程中水母"放电"
- **两个模式**: 加密模式 / 解密模式，底部切换
- **支持**: 粘贴加密密文后一键解密

### 4.4 🦀 痞老板的爱意值计算器
- **角色**: 痞老板（Plankton）
- **玩法**: 输入两人的名字 → 点击计算 → 显示爱意值百分比+搞笑评语
- **视觉**: 痞老板在实验台前，实验瓶冒泡，结果弹出
- **逻辑**: 名字字符串哈希 → 映射到 60%-100% 范围 → 匹配对应评语
- **相同输入→相同结果**，可反复测试不同名字组合
- **评语分级**: 60-70% / 71-80% / 81-90% / 91-99% / 100% 各档有不同搞怪文案

## 5. 技术实现

- **技术栈**: 纯 HTML/CSS/JS，零依赖，单文件
- **响应式**: 手机端优先（max-width: 480px 居中），PC 端显示手机模拟框
- **动画**: CSS animation + JS 触发动画
- **存储**: 无后端，无 localStorage 依赖
- **兼容**: 现代浏览器（Chrome, Safari, 微信内置浏览器）

## 6. 交互细节

- Tab 切换：点击底部图标切换，带弹性过渡 (spring-like ease)
- 按钮：按压缩放反馈 (transform: scale(0.92))
- 气泡上升：随机位置、大小、速度，纯 CSS + JS 生成
- 海草摇摆：CSS keyframes，不同海草不同延迟
- 页面加载：标题从海底"浮上来"的入场动画
